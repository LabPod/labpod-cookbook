# PyTorch GPU Basics (Cookbook)

Pull-only environment - `quay.io/jupyter/pytorch-notebook:cuda12-python-3.11`, no custom build.
Same image as LabPod's built-in "PyTorch JupyterLab" template, packaged standalone so this
cookbook works even on a LabPod install where that managed template is disabled.

After building and starting a workspace from this template, get the notebook:

```bash
git clone https://github.com/LabPod/labpod-cookbook /work/labpod-cookbook
```

Then open `/work/labpod-cookbook/pytorch-gpu-basics/notebook.ipynb` in JupyterLab.
